# Hotel-Price-Prediction
Machine learning project to predict hotel prices using data preprocessing, feature engineering, and model evaluation

## Project Structure
- eda.ipynb → Exploratory Data Analysis  
- model.ipynb → Model training and evaluation  

## Features
- Data preprocessing  
- Feature engineering  
- Model building and evaluation  

## Tech Stack
Python, Pandas, NumPy, Scikit-learn
